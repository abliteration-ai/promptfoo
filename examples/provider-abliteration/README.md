# provider-abliteration (Abliteration)

You can run this example with:

```bash
npx promptfoo@latest init --example provider-abliteration
cd provider-abliteration
```

## Usage

Set your `ABLIT_KEY` environment variable and replace `abliterated-model` in `promptfooconfig.yaml` with your Abliteration model ID.

Then run:

```bash
promptfoo eval
```
